# 教师端指导记录接口文档

## 1. 获取指导统计数据

### 接口信息
- **接口路径**: `/api/teacher/guidance/stats`
- **请求方式**: `GET`
- **接口描述**: 获取教师的指导统计数据

### 请求参数
无

### 请求头
```
Authorization: Bearer {token}
```

### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "totalCount": 45,
    "projectCount": 12,
    "monthCount": 8
  }
}
```

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| totalCount | Number | 总指导次数 |
| projectCount | Number | 指导项目数（去重） |
| monthCount | Number | 本月指导次数 |

### 数据库查询示例
```sql
-- 总指导次数
SELECT COUNT(*) as total_count
FROM guidance
WHERE teacher_id = #{teacherId}

-- 指导项目数
SELECT COUNT(DISTINCT project_id) as project_count
FROM guidance
WHERE teacher_id = #{teacherId}

-- 本月指导次数
SELECT COUNT(*) as month_count
FROM guidance
WHERE teacher_id = #{teacherId}
  AND DATE_FORMAT(create_time, '%Y-%m') = DATE_FORMAT(NOW(), '%Y-%m')
```

## 2. 获取指导记录列表

### 接口信息
- **接口路径**: `/api/teacher/guidance/list`
- **请求方式**: `GET`
- **接口描述**: 获取教师的指导记录列表，支持搜索和分页

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | Number | 是 | 页码，从1开始 |
| pageSize | Number | 是 | 每页数量，建议10 |
| keyword | String | 否 | 搜索关键词（项目名称） |

### 请求头
```
Authorization: Bearer {token}
```

### 请求示例

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| total | Number | 总记录数 |

#### list数组元素说明
| 字段名 | 类型 | 说明 |
|--------|------|------|

| teacherId | String | 教师ID |
| teacherName | String | 教师姓名 |
| teacherAvatar | String | 教师头像URL |
| content | String | 指导内容 |
| rating | Number | 评分（可选，满分5分） |
| createTime | String | 指导时间（相对时间） |

### 业务逻辑说明
1. 只查询当前登录教师的指导记录
2. 支持按项目名称模糊搜索
3. 按指导时间倒序排列

### 数据库查询示例
```sql
-- 查询指导记录列表
SELECT 
  g.guidance_id as id,
  g.project_id,
  p.project_name,
  p.logo as project_logo,
  g.student_id,
  u.nickname as student_name,
  p.domain,
  g.teacher_id,
  t.name as teacher_name,
  t.avatar as teacher_avatar,
  g.content,
  g.rating,
  g.create_time
FROM guidance g
LEFT JOIN project p ON g.project_id = p.project_id
LEFT JOIN user u ON g.student_id = u.user_id
LEFT JOIN teacher t ON g.teacher_id = t.teacher_id
WHERE g.teacher_id = #{teacherId}
  AND (#{keyword} IS NULL OR p.project_name LIKE CONCAT('%', #{keyword}, '%'))
ORDER BY g.create_time DESC
LIMIT #{offset}, #{pageSize}

-- 查询总数
SELECT COUNT(*) as total
FROM guidance g
LEFT JOIN project p ON g.project_id = p.project_id
WHERE g.teacher_id = #{teacherId}
  AND (#{keyword} IS NULL OR p.project_name LIKE CONCAT('%', #{keyword}, '%'))
```

## 3. 发表指导意见

### 接口信息
- **接口路径**: `/api/teacher/guidance/add`
- **请求方式**: `POST`
- **接口描述**: 教师对项目发表指导意见

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectId | String | 是 | 项目ID |
| content | String | 是 | 指导内容，不少于10个字 |

### 请求头
```
Authorization: Bearer {token}
Content-Type: application/json
```

### 请求示例
```json
{
  "projectId": "1",
  "content": "项目方向很好，建议加强技术实现的可行性分析，特别是在数据安全和隐私保护方面需要更详细的方案。"
}
```

### 响应数据
```json
{
  "code": 200,
  "message": "指导意见发表成功",
  "data": {
    "guidanceId": "123"
  }
}
```

### 响应字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| guidanceId | String | 新创建的指导记录ID |

### 业务逻辑说明
1. 从token中解析出教师ID
2. 验证项目是否存在
3. 验证指导内容长度（不少于10个字）
4. 插入指导记录
5. 更新项目的指导次数
6. 可选：给学生发送通知

### 数据库操作示例
```sql
-- 1. 验证项目是否存在
SELECT project_id, student_id FROM project WHERE project_id = #{projectId}

-- 2. 插入指导记录
INSERT INTO guidance (
  guidance_id,
  project_id,
  student_id,
  teacher_id,
  content,
  create_time
) VALUES (
  #{guidanceId},
  #{projectId},
  #{studentId},
  #{teacherId},
  #{content},
  NOW()
)

-- 3. 更新项目的指导次数（可选，如果有这个字段）
UPDATE project 
SET guidance_count = guidance_count + 1
WHERE project_id = #{projectId}
```

### 错误码说明
| 错误码 | 说明 |
|--------|------|
| 400 | 参数错误（项目ID为空、内容过短等） |
| 401 | 未登录或token过期 |
| 403 | 无权限访问（非教师角色） |
| 404 | 项目不存在 |
| 500 | 服务器内部错误 |

### 注意事项
1. 该接口需要验证用户角色为教师（role = 'ROLE_TEACHER'）
2. 指导内容不能为空，且不少于10个字
3. 同一个教师可以对同一个项目多次指导
4. 建议记录指导时间，方便后续统计和展示

### 前端调用示例
```typescript
import { addGuidance } from '@/api/teacher'

const handleSubmit = async () => {
  try {
    await addGuidance({
      projectId: '1',
      content: '项目方向很好，建议加强技术实现...'
    })
    
    showToast('指导意见发表成功')
    // 刷新页面或返回上一页
  } catch (error) {
    console.error('发表失败', error)
  }
}
```
