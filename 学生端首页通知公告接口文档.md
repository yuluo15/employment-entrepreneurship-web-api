# 学生端首页通知公告接口文档

## 功能概述

在学生端首页展示管理员和学校发布的通知公告，让学生及时了解重要信息。

---

## 接口设计

## SQL查询示例

### 1. 首页通知列表（最新5条）
```sql
SELECT 
  n.notice_id,
  n.notice_title,
  n.notice_type,
  CASE n.notice_type
    WHEN 'ACTIVITY' THEN '活动通知'
    WHEN 'COMPETITION' THEN '赛事通知'
    WHEN 'LECTURE' THEN '讲座通知'
    WHEN 'POLICY' THEN '政策通知'
    ELSE '一般通知'
  END as notice_type_text,
  n.publish_time,
  n.publisher_type,
  CASE 
    WHEN n.publisher_type = 'ADMIN' THEN '省教育厅'
    WHEN n.publisher_type = 'SCHOOL' THEN s.name
    ELSE '未知'
  END as publisher_name,
  n.is_top,
  n.view_count
FROM sys_notice n
LEFT JOIN sys_school s ON n.publisher_id = s.id AND n.publisher_type = 'SCHOOL'
WHERE n.status = 1  -- 已发布
  AND (
    n.target_audience = 'ALL'  -- 面向所有人
    OR n.target_audience = 'STUDENT'  -- 面向学生
    OR (n.target_audience = 'SCHOOL' AND n.publisher_id = #{studentSchoolId})  -- 本校通知
  )
ORDER BY n.is_top DESC, n.publish_time DESC
LIMIT 5
```

### 2. 通知详情
```sql
-- 查询详情
SELECT 
  n.*,
  CASE n.notice_type
    WHEN 'ACTIVITY' THEN '活动通知'
    WHEN 'COMPETITION' THEN '赛事通知'
    WHEN 'LECTURE' THEN '讲座通知'
    WHEN 'POLICY' THEN '政策通知'
    ELSE '一般通知'
  END as notice_type_text,
  CASE 
    WHEN n.publisher_type = 'ADMIN' THEN '省教育厅'
    WHEN n.publisher_type = 'SCHOOL' THEN s.name
    ELSE '未知'
  END as publisher_name
FROM sys_notice n
LEFT JOIN sys_school s ON n.publisher_id = s.id AND n.publisher_type = 'SCHOOL'
WHERE n.notice_id = #{noticeId}
  AND n.status = 1

-- 更新浏览次数
UPDATE sys_notice
SET view_count = view_count + 1
WHERE notice_id = #{noticeId}
```

### 3. 完整列表（分页）
```sql
SELECT 
  n.notice_id,
  n.notice_title,
  n.notice_type,
  CASE n.notice_type
    WHEN 'ACTIVITY' THEN '活动通知'
    WHEN 'COMPETITION' THEN '赛事通知'
    WHEN 'LECTURE' THEN '讲座通知'
    WHEN 'POLICY' THEN '政策通知'
    ELSE '一般通知'
  END as notice_type_text,
  n.publish_time,
  n.publisher_type,
  CASE 
    WHEN n.publisher_type = 'ADMIN' THEN '省教育厅'
    WHEN n.publisher_type = 'SCHOOL' THEN s.name
    ELSE '未知'
  END as publisher_name,
  n.is_top,
  n.view_count
FROM sys_notice n
LEFT JOIN sys_school s ON n.publisher_id = s.id AND n.publisher_type = 'SCHOOL'
WHERE n.status = 1
  AND (
    n.target_audience = 'ALL'
    OR n.target_audience = 'STUDENT'
    OR (n.target_audience = 'SCHOOL' AND n.publisher_id = #{studentSchoolId})
  )
  AND (#{noticeType} IS NULL OR n.notice_type = #{noticeType})
  AND (#{publisherType} IS NULL OR n.publisher_type = #{publisherType})
ORDER BY n.is_top DESC, n.publish_time DESC
LIMIT #{offset}, #{pageSize}
```

## 权限说明

### 学生可见的通知范围

1. **省级通知**：`publisher_type = 'ADMIN'` 且 `target_audience = 'ALL'` 或 `'STUDENT'`
2. **本校通知**：`publisher_type = 'SCHOOL'` 且 `publisher_id = 学生所在学校ID`
3. **状态**：只显示已发布的通知（`status = 1`）

## 注意事项

1. **权限控制**：
   - 学生只能看到面向学生的通知
   - 学校通知只显示本校的

2. **置顶逻辑**：
   - 置顶通知优先显示
   - 同为置顶按发布时间倒序

3. **浏览统计**：
   - 每次查看详情时浏览次数+1
   - 可以考虑防刷机制（同一用户短时间内多次查看不重复计数）

4. **内容安全**：
   - 通知内容为HTML格式，前端需要做XSS防护
   - 使用 `v-html` 时要确保内容安全
