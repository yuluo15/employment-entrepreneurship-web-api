# 教师端工作台接口文档

## 1. 获取教师工作台数据

### 接口信息
- **接口路径**: `/api/teacher/dashboard`
- **请求方式**: `GET`
- **接口描述**: 获取教师工作台的所有数据，包括教师信息、统计数据、待指导项目、最近指导记录

### 请求参数
无

### 请求头
```
Authorization: Bearer {token}
```

### 业务逻辑说明

1. **教师信息获取**
   - 从token中解析出当前登录的教师ID
   - 查询教师基本信息（姓名、职称、学院等）

2. **统计数据计算**
   - `guidanceCount`: 统计该教师的总指导次数
   - `projectCount`: 统计该教师指导过的项目数量（去重）
   - `ratingScore`: 计算该教师的平均评分
   - `weekNewCount`: 统计本周新增的本校项目数量

3. **待指导项目**
   - 查询指导次数为0的项目
   - 优先显示本校项目
   - 按创建时间倒序排列
   - 最多返回5条

4. **最近指导记录**
   - 查询该教师最近的指导记录
   - 按指导时间倒序排列
   - 最多返回5条

### 数据库查询示例

```sql
-- 1. 获取教师信息
SELECT 
  t.teacher_id,
  t.name,
  t.avatar,
  t.title,
  t.college_name,
  t.school_id,
  s.school_name,
  t.expertise
FROM teacher t
LEFT JOIN school s ON t.school_id = s.school_id
WHERE t.teacher_id = #{teacherId}

-- 2. 统计指导次数
SELECT COUNT(*) as guidance_count
FROM guidance
WHERE teacher_id = #{teacherId}

-- 3. 统计指导项目数
SELECT COUNT(DISTINCT project_id) as project_count
FROM guidance
WHERE teacher_id = #{teacherId}

-- 4. 计算平均评分
SELECT AVG(rating) as rating_score
FROM guidance
WHERE teacher_id = #{teacherId} AND rating IS NOT NULL

-- 5. 统计本周新增项目数
SELECT COUNT(*) as week_new_count
FROM project p
WHERE p.school_id = (SELECT school_id FROM teacher WHERE teacher_id = #{teacherId})
  AND p.create_time >= DATE_SUB(NOW(), INTERVAL 7 DAY)

-- 6. 查询待指导项目
SELECT 
  p.project_id,
  p.project_name,
  p.logo,
  p.student_id,
  u.nickname as student_name,
  p.domain,
  p.create_time
FROM project p
LEFT JOIN user u ON p.student_id = u.user_id
WHERE p.project_id NOT IN (
  SELECT DISTINCT project_id FROM guidance WHERE teacher_id = #{teacherId}
)
ORDER BY p.create_time DESC
LIMIT 5

-- 7. 查询最近指导记录
SELECT 
  g.guidance_id as id,
  g.project_id,
  p.project_name,
  g.content,
  g.create_time
FROM guidance g
LEFT JOIN project p ON g.project_id = p.project_id
WHERE g.teacher_id = #{teacherId}
ORDER BY g.create_time DESC
LIMIT 5
```

### 错误码说明
| 错误码 | 说明 |
|--------|------|
| 401 | 未登录或token过期 |
| 403 | 无权限访问（非教师角色） |
| 500 | 服务器内部错误 |

### 注意事项
1. 该接口需要验证用户角色为教师（role = 'ROLE_TEACHER'）
2. 时间字段需要转换为相对时间格式（如"2天前"、"1小时前"）
3. 评分保留一位小数
4. 待指导项目和最近指导记录最多返回5条，前端可以点击"查看全部"跳转到对应页面

### 前端调用示例
```typescript
import { getTeacherDashboard } from '@/api/teacher'

const loadData = async () => {
  try {
    const res = await getTeacherDashboard()
    const data = res.data
    
    // 使用返回的数据更新页面
    teacherInfo.value = data.teacherInfo
    stats.value = data.stats
    pendingProjects.value = data.pendingProjects
    recentGuidance.value = data.recentGuidance
  } catch (error) {
    console.error('加载数据失败', error)
  }
}
```
