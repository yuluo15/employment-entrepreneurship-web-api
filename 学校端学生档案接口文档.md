# 学校端学生档案接口文档

## 1. 获取学生列表

**接口地址**: `GET /api/school/student/list`

**请求头**:
```
Authorization: Bearer {token}
```

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | int | 是 | 页码 |
| pageSize | int | 是 | 每页数量 |
| studentName | string | 否 | 学生姓名（模糊查询） |
| studentNo | string | 否 | 学号（模糊查询） |
| collegeName | string | 否 | 学院名称（模糊查询） |
| majorName | string | 否 | 专业名称（模糊查询） |
| enrollmentYear | int | 否 | 入学年份 |
| employmentStatus | string | 否 | 就业状态（0=待就业, 1=已就业, 2=升学, 3=出国, 4=创业） |

**响应示例**:
```json
{
  "message": "success",
  "data": {
    "total": 150,
    "data": [
      {
        "studentId": "stu123",
        "userId": "user456",
        "schoolId": "school001",
        "studentName": "张三",
        "studentNo": "2021001",
        "gender": "1",
        "collegeName": "计算机学院",
        "majorName": "软件工程",
        "className": "软件2101",
        "education": "本科",
        "enrollmentYear": 2021,
        "graduationYear": 2025,
        "phone": "13800138000",
        "email": "zhangsan@example.com",
        "employmentStatus": "1",
        "createTime": "2021-09-01 10:00:00"
      }
    ]
  }
}
```

**字段说明**:
- `gender`: 1=男, 2=女
- `employmentStatus`: 0=待就业, 1=已就业
- `education`: 本科、硕士、博士等
- **就业状态**: 企业发放offer时自动设置为"已就业"
- **创业项目**: 学生参与的创业项目通过 `biz_project` 表管理，不影响就业状态

## 4. 下载导入模板

**接口地址**: `GET /api/school/student/template`

**请求头**:
```
Authorization: Bearer {token}
```

**请求参数**: 无

**响应**: Excel文件流

**说明**: 
- 返回学生信息导入模板
- 文件名：`学生信息导入模板.xlsx`
- 模板包含以下列：
  - 学号（必填）
  - 姓名（必填）
  - 性别（必填，填写：男 或 女）
  - 学院（必填）
  - 专业（必填）
  - 班级（选填）
  - 学历（必填，填写：专科、本科、硕士、博士）
  - 入学年份（必填，如：2021）
  - 毕业年份（必填，如：2025）
  - 联系电话（选填）
  - 邮箱（选填）

## 5. 批量导入学生

**接口地址**: `POST /api/school/student/import`

**请求头**:
```
Authorization: Bearer {token}
Content-Type: multipart/form-data
```

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| file | File | 是 | Excel文件 |

**响应示例**（全部成功）:
```json
{
  "message": "success",
  "data": {
    "successCount": 50,
    "failCount": 0,
    "failList": []
  }
}
```

**响应示例**（部分失败）:
```json
{
  "message": "success",
  "data": {
    "successCount": 48,
    "failCount": 2,
    "failList": [
      {
        "row": 3,
        "studentNo": "2021003",
        "reason": "学号已存在"
      },
      {
        "row": 15,
        "studentNo": "2021015",
        "reason": "性别格式错误，应填写：男 或 女"
      }
    ]
  }
}
```

**字段说明**:
- `successCount`: 成功导入的学生数量
- `failCount`: 导入失败的学生数量
- `failList`: 失败记录列表
  - `row`: Excel中的行号
  - `studentNo`: 学号
  - `reason`: 失败原因

**导入规则**:
1. 系统会自动为每个学生创建账号（`sys_user`表）和学生档案（`biz_student`表）
2. 账号的 `login_identity` 为学号
3. 初始密码统一为：`123456`（需要MD5加密存储）
4. `role_key` 设置为 `student`
5. `owner_id` 设置为当前学校ID
6. 性别转换：男=1，女=2
7. 如果学号已存在，跳过该条记录并记录失败原因
8. 如果必填字段缺失，跳过该条记录并记录失败原因

### Service实现示例

```java
@Service
public class SchoolStudentServiceImpl implements SchoolStudentService {
    
    @Autowired
    private SysUserMapper userMapper;
    
    @Autowired
    private BizStudentMapper studentMapper;
    
    @Transactional(rollbackFor = Exception.class)
    public ImportResultVO importStudents(MultipartFile file, String schoolId) throws IOException {
        ImportResultVO result = new ImportResultVO();
        int successCount = 0;
        int failCount = 0;
        List<ImportFailItem> failList = new ArrayList<>();
        
        // 读取Excel
        List<StudentImportDTO> dataList = EasyExcel.read(file.getInputStream())
            .head(StudentImportDTO.class)
            .sheet()
            .doReadSync();
        
        for (int i = 0; i < dataList.size(); i++) {
            StudentImportDTO dto = dataList.get(i);
            int rowNum = i + 2; // Excel行号（从第2行开始，第1行是表头）
            
            try {
                // 1. 数据校验
                String validateError = validateStudentData(dto);
                if (validateError != null) {
                    failList.add(new ImportFailItem(rowNum, dto.getStudentNo(), validateError));
                    failCount++;
                    continue;
                }
                
                // 2. 检查学号是否已存在
                if (studentMapper.existsByStudentNo(dto.getStudentNo())) {
                    failList.add(new ImportFailItem(rowNum, dto.getStudentNo(), "学号已存在"));
                    failCount++;
                    continue;
                }
                
                // 3. 创建用户账号
                SysUser user = new SysUser();
                user.setId(UUID.randomUUID().toString().replace("-", ""));
                user.setLoginIdentity(dto.getStudentNo()); // 学号作为登录账号
                user.setPassword(DigestUtils.md5Hex("123456")); // 初始密码
                user.setRealName(dto.getStudentName());
                user.setRoleKey("student");
                user.setOwnerId(schoolId);
                user.setStatus(1);
                user.setIsDeleted(0);
                user.setCreateTime(new Date());
                user.setUpdateTime(new Date());
                userMapper.insert(user);
                
                // 4. 创建学生档案
                BizStudent student = new BizStudent();
                student.setStudentId(UUID.randomUUID().toString().replace("-", ""));
                student.setUserId(user.getId());
                student.setSchoolId(schoolId);
                student.setStudentName(dto.getStudentName());
                student.setStudentNo(dto.getStudentNo());
                student.setGender("男".equals(dto.getGender()) ? "1" : "2");
                student.setCollegeName(dto.getCollegeName());
                student.setMajorName(dto.getMajorName());
                student.setClassName(dto.getClassName());
                student.setEducation(dto.getEducation());
                student.setEnrollmentYear(dto.getEnrollmentYear());
                student.setGraduationYear(dto.getGraduationYear());
                student.setPhone(dto.getPhone());
                student.setEmail(dto.getEmail());
                student.setEmploymentStatus("0"); // 默认待就业
                student.setCreateTime(new Date());
                student.setUpdateTime(new Date());
                studentMapper.insert(student);
                
                successCount++;
                
            } catch (Exception e) {
                log.error("导入第{}行失败", rowNum, e);
                failList.add(new ImportFailItem(rowNum, dto.getStudentNo(), "系统错误：" + e.getMessage()));
                failCount++;
            }
        }
        
        result.setSuccessCount(successCount);
        result.setFailCount(failCount);
        result.setFailList(failList);
        
        return result;
    }
    
    private String validateStudentData(StudentImportDTO dto) {
        if (StringUtils.isBlank(dto.getStudentNo())) {
            return "学号不能为空";
        }
        if (StringUtils.isBlank(dto.getStudentName())) {
            return "姓名不能为空";
        }
        if (StringUtils.isBlank(dto.getGender())) {
            return "性别不能为空";
        }
        if (!"男".equals(dto.getGender()) && !"女".equals(dto.getGender())) {
            return "性别格式错误，应填写：男 或 女";
        }
        if (StringUtils.isBlank(dto.getCollegeName())) {
            return "学院不能为空";
        }
        if (StringUtils.isBlank(dto.getMajorName())) {
            return "专业不能为空";
        }
        if (StringUtils.isBlank(dto.getEducation())) {
            return "学历不能为空";
        }
        if (!Arrays.asList("专科", "本科", "硕士", "博士").contains(dto.getEducation())) {
            return "学历格式错误，应填写：专科、本科、硕士、博士";
        }
        if (dto.getEnrollmentYear() == null) {
            return "入学年份不能为空";
        }
        if (dto.getGraduationYear() == null) {
            return "毕业年份不能为空";
        }
        if (dto.getEnrollmentYear() >= dto.getGraduationYear()) {
            return "毕业年份必须大于入学年份";
        }
        return null;
    }
}
```

### Java Controller示例

```java
@RestController
@RequestMapping("/api/school/student")
public class SchoolStudentController {

    @Autowired
    private SchoolStudentService studentService;

    @GetMapping("/list")
    public Result<PageResult<StudentVO>> getStudentList(StudentQuery query) {
        // 从token获取当前学校ID
        String schoolId = SecurityUtils.getSchoolId();
        query.setSchoolId(schoolId);
        
        PageResult<StudentVO> result = studentService.getStudentList(query);
        return Result.success(result);
    }

    @GetMapping("/detail/{studentId}")
    public Result<StudentVO> getStudentDetail(@PathVariable String studentId) {
        String schoolId = SecurityUtils.getSchoolId();
        StudentVO student = studentService.getStudentDetail(studentId, schoolId);
        
        if (student == null) {
            return Result.error("学生不存在或无权访问");
        }
        
        return Result.success(student);
    }

    @GetMapping("/resume/{studentId}")
    public Result<StudentResumeVO> getStudentResume(@PathVariable String studentId) {
        String schoolId = SecurityUtils.getSchoolId();
        StudentResumeVO resume = studentService.getStudentResume(studentId, schoolId);
        
        if (resume == null) {
            return Result.error(404, "该学生暂未完善简历");
        }
        
        return Result.success(resume);
    }

    @GetMapping("/export")
    public void exportStudentData(StudentQuery query, HttpServletResponse response) {
        String schoolId = SecurityUtils.getSchoolId();
        query.setSchoolId(schoolId);
        
        studentService.exportStudentData(query, response);
    }
}
```

## 注意事项

1. **权限控制**: 学校端只能查看本校学生数据，需要在SQL中加上 `school_id` 过滤条件
2. **数据隔离**: 从token中获取当前登录用户所属的学校ID，确保数据隔离
3. **简历字段**: `education_history`、`internship_exp`、`project_exp`、`certificates` 是JSONB类型，需要正确解析
4. **就业状态**: 0=待就业, 1=已就业, 2=升学, 3=出国, 4=创业
5. **性别字段**: 1=男, 2=女
6. **导出功能**: 使用Apache POI或EasyExcel实现Excel导出
7. **简历不存在**: 如果学生未完善简历，返回404状态码
8. **批量导入**:
   - 使用EasyExcel读取Excel文件
   - 需要同时创建 `sys_user` 和 `biz_student` 两张表的记录
   - 使用事务确保数据一致性
   - 初始密码为 `123456`，需要MD5加密
   - 学号作为登录账号（`login_identity`）
   - 性别转换：男=1，女=2
   - 导入失败的记录需要返回详细的失败原因
9. **模板下载**: 模板中应包含示例数据，方便用户理解填写格式
10. **文件大小限制**: 建议限制上传文件大小不超过5MB
11. **并发控制**: 如果导入数据量大，建议使用异步处理或分批导入
