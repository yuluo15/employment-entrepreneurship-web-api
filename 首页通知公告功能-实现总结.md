## 需要完成的后端接口

### 接口1：获取首页通知列表

**接口路径**: `/api/mobile/home/notices`  
**请求方式**: `GET`  
**接口描述**: 返回最新5条通知公告

#### 响应数据
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "noticeId": "1",
      "noticeTitle": "关于举办2026届秋季校园双选会的通知",
      "noticeType": "ACTIVITY",
      "noticeTypeText": "活动通知",
      "publishTime": "2024-01-15 10:30:00",
      "publisherType": "ADMIN",
      "publisherName": "省教育厅",
      "isTop": 1,
      "viewCount": 1250
    }
  ]
}
```

#### SQL查询
```sql
SELECT 
  n.notice_id,
  n.notice_title,
  n.notice_type,
  CASE n.notice_type
    WHEN 'ACTIVITY' THEN '活动通知'
    WHEN 'COMPETITION' THEN '赛事通知'
    WHEN 'LECTURE' THEN '讲座通知'
    WHEN 'POLICY' THEN '政策通知'
    ELSE '一般通知'
  END as notice_type_text,
  n.publish_time,
  n.publisher_type,
  CASE 
    WHEN n.publisher_type = 'ADMIN' THEN '省教育厅'
    WHEN n.publisher_type = 'SCHOOL' THEN s.name
    ELSE '未知'
  END as publisher_name,
  n.is_top,
  n.view_count
FROM sys_notice n
LEFT JOIN sys_school s ON n.publisher_id = s.id AND n.publisher_type = 'SCHOOL'
WHERE n.status = 1  -- 已发布
  AND (
    n.target_audience = 'ALL'  -- 面向所有人
    OR n.target_audience = 'STUDENT'  -- 面向学生
    OR (n.target_audience = 'SCHOOL' AND n.publisher_id = #{studentSchoolId})  -- 本校通知
  )
ORDER BY n.is_top DESC, n.publish_time DESC
LIMIT 5
```

#### 权限说明
学生可见的通知范围：
1. 省级通知：`publisher_type = 'ADMIN'` 且 `target_audience = 'ALL'` 或 `'STUDENT'`
2. 本校通知：`publisher_type = 'SCHOOL'` 且 `publisher_id = 学生所在学校ID`
3. 状态：只显示已发布的通知（`status = 1`）

**接口路径**: `/api/mobile/notices`  
**请求方式**: `GET`  
**接口描述**: 获取所有通知列表，支持分页和筛选

#### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | Number | 是 | 页码 |

| publisherType | String | 否 | 发布者类型筛选 |

---

## 通知类型说明

| 类型代码 | 类型名称 | 标签颜色 |
|---------|---------|---------|
| ACTIVITY | 活动通知 | 蓝色(primary) |
| COMPETITION | 赛事通知 | 橙色(warning) |
| LECTURE | 讲座通知 | 绿色(success) |
| POLICY | 政策通知 | 红色(danger) |
| GENERAL | 一般通知 | 灰色(default) |

---

## 数据库表结构

使用现有的 `sys_notice` 表，字段说明：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| notice_id | varchar(64) | 通知ID |
| notice_title | varchar(200) | 通知标题 |
| notice_type | varchar(10) | 通知类型 |
| notice_content | text | 通知内容（HTML） |
| status | int2 | 状态：0=草稿，1=已发布 |
| is_top | int2 | 是否置顶：0=否，1=是 |
| view_count | int4 | 浏览次数 |
| publish_time | timestamp | 发布时间 |
| publisher_type | varchar(255) | 发布者类型：ADMIN/SCHOOL |
| publisher_id | varchar(255) | 发布者ID |
| target_audience | varchar(255) | 目标受众：ALL/STUDENT/SCHOOL |

## 功能特点

1. **实时更新**：展示最新5条通知
2. **类型标签**：不同类型用不同颜色标签
3. **置顶标识**：置顶通知显示火焰图标
4. **浏览统计**：显示浏览次数
5. **权限控制**：只显示学生可见的通知
6. **空状态**：没有通知时显示空状态提示
