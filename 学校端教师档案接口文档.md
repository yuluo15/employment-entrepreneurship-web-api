# 学校端教师档案接口文档

## 1. 获取教师列表

**接口地址**: `GET /api/school/teacher/list`

**请求头**:
```
Authorization: Bearer {token}
```

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | int | 是 | 页码 |
| pageSize | int | 是 | 每页数量 |
| name | string | 否 | 教师姓名（模糊查询） |
| employeeNo | string | 否 | 工号（模糊查询） |
| collegeName | string | 否 | 学院名称（模糊查询） |
| title | string | 否 | 职称（模糊查询） |

**响应示例**:
```json
{
  "message": "success",
  "data": {
    "total": 80,
    "data": [
      {
        "teacherId": "teacher123",
        "userId": "user789",
        "schoolId": "school001",
        "name": "王教授",
        "employeeNo": "T2020001",
        "gender": "1",
        "collegeName": "计算机学院",
        "title": "教授",
        "expertise": "人工智能、机器学习",
        "intro": "从事人工智能研究20年，发表论文50余篇...",
        "guidanceCount": 15,
        "ratingScore": 4.8,
        "phone": "13900139000",
        "email": "wangprof@example.com",
        "createTime": "2020-09-01 10:00:00"
      }
    ]
  }
}
```

**字段说明**:
- `gender`: 1=男, 2=女
- `title`: 教授、副教授、讲师、助教等
- `expertise`: 专业领域，多个领域用逗号分隔
- `guidanceCount`: 指导学生数量
- `ratingScore`: 评分（0-5分，保留1位小数）

## 3. 下载导入模板

**接口地址**: `GET /api/school/teacher/template`

**请求头**:
```
Authorization: Bearer {token}
```

**请求参数**: 无

**响应**: Excel文件流

**说明**: 
- 返回教师信息导入模板
- 文件名：`教师信息导入模板.xlsx`
- 模板包含以下列：
  - 工号（必填）
  - 姓名（必填）
  - 性别（必填，填写：男 或 女）
  - 学院（必填）
  - 职称（必填，填写：教授、副教授、讲师、助教）
  - 专业领域（选填，多个领域用逗号分隔）
  - 联系电话（选填）
  - 邮箱（选填）
  - 个人简介（选填）

## 4. 批量导入教师

**接口地址**: `POST /api/school/teacher/import`

**请求头**:
```
Authorization: Bearer {token}
Content-Type: multipart/form-data
```

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| file | File | 是 | Excel文件 |

**响应示例**（全部成功）:
```json
{
  "message": "success",
  "data": {
    "successCount": 30,
    "failCount": 0,
    "failList": []
  }
}
```

**响应示例**（部分失败）:
```json
{
  "message": "success",
  "data": {
    "successCount": 28,
    "failCount": 2,
    "failList": [
      {
        "row": 3,
        "employeeNo": "T2020003",
        "reason": "工号已存在"
      },
      {
        "row": 10,
        "employeeNo": "T2020010",
        "reason": "职称格式错误，应填写：教授、副教授、讲师、助教"
      }
    ]
  }
}
```

**字段说明**:
- `successCount`: 成功导入的教师数量
- `failCount`: 导入失败的教师数量
- `failList`: 失败记录列表
  - `row`: Excel中的行号
  - `employeeNo`: 工号
  - `reason`: 失败原因

**导入规则**:
1. 系统会自动为每个教师创建账号（`sys_user`表）和教师档案（`biz_teacher`表）
2. 账号的 `login_identity` 为工号
3. 初始密码统一为：`123456`（需要MD5加密存储）
4. `role_key` 设置为 `teacher`
5. `owner_id` 设置为当前学校ID
6. 性别转换：男=1，女=2
7. 如果工号已存在，跳过该条记录并记录失败原因
8. 如果必填字段缺失，跳过该条记录并记录失败原因
9. `guidance_count` 初始值为 0
10. `rating_score` 初始值为 5.0

### Service实现示例

```java
@Service
public class SchoolTeacherServiceImpl implements SchoolTeacherService {
    
    @Autowired
    private SysUserMapper userMapper;
    
    @Autowired
    private BizTeacherMapper teacherMapper;
    
    @Transactional(rollbackFor = Exception.class)
    public ImportResultVO importTeachers(MultipartFile file, String schoolId) throws IOException {
        ImportResultVO result = new ImportResultVO();
        int successCount = 0;
        int failCount = 0;
        List<ImportFailItem> failList = new ArrayList<>();
        
        // 读取Excel
        List<TeacherImportDTO> dataList = EasyExcel.read(file.getInputStream())
            .head(TeacherImportDTO.class)
            .sheet()
            .doReadSync();
        
        for (int i = 0; i < dataList.size(); i++) {
            TeacherImportDTO dto = dataList.get(i);
            int rowNum = i + 2; // Excel行号（从第2行开始，第1行是表头）
            
            try {
                // 1. 数据校验
                String validateError = validateTeacherData(dto);
                if (validateError != null) {
                    failList.add(new ImportFailItem(rowNum, dto.getEmployeeNo(), validateError));
                    failCount++;
                    continue;
                }
                
                // 2. 检查工号是否已存在
                if (teacherMapper.existsByEmployeeNo(dto.getEmployeeNo())) {
                    failList.add(new ImportFailItem(rowNum, dto.getEmployeeNo(), "工号已存在"));
                    failCount++;
                    continue;
                }
                
                // 3. 创建用户账号
                SysUser user = new SysUser();
                user.setId(UUID.randomUUID().toString().replace("-", ""));
                user.setLoginIdentity(dto.getEmployeeNo()); // 工号作为登录账号
                user.setPassword(DigestUtils.md5Hex("123456")); // 初始密码
                user.setRealName(dto.getName());
                user.setRoleKey("teacher");
                user.setOwnerId(schoolId);
                user.setStatus(1);
                user.setIsDeleted(0);
                user.setCreateTime(new Date());
                user.setUpdateTime(new Date());
                userMapper.insert(user);
                
                // 4. 创建教师档案
                BizTeacher teacher = new BizTeacher();
                teacher.setTeacherId(UUID.randomUUID().toString().replace("-", ""));
                teacher.setUserId(user.getId());
                teacher.setSchoolId(schoolId);
                teacher.setName(dto.getName());
                teacher.setEmployeeNo(dto.getEmployeeNo());
                teacher.setGender("男".equals(dto.getGender()) ? "1" : "2");
                teacher.setCollegeName(dto.getCollegeName());
                teacher.setTitle(dto.getTitle());
                teacher.setExpertise(dto.getExpertise());
                teacher.setIntro(dto.getIntro());
                teacher.setPhone(dto.getPhone());
                teacher.setEmail(dto.getEmail());
                teacher.setGuidanceCount(0); // 初始指导数量为0
                teacher.setRatingScore(new BigDecimal("5.0")); // 初始评分5.0
                teacher.setCreateTime(new Date());
                teacher.setUpdateTime(new Date());
                teacherMapper.insert(teacher);
                
                successCount++;
                
            } catch (Exception e) {
                log.error("导入第{}行失败", rowNum, e);
                failList.add(new ImportFailItem(rowNum, dto.getEmployeeNo(), "系统错误：" + e.getMessage()));
                failCount++;
            }
        }
        
        result.setSuccessCount(successCount);
        result.setFailCount(failCount);
        result.setFailList(failList);
        
        return result;
    }
    
    private String validateTeacherData(TeacherImportDTO dto) {
        if (StringUtils.isBlank(dto.getEmployeeNo())) {
            return "工号不能为空";
        }
        if (StringUtils.isBlank(dto.getName())) {
            return "姓名不能为空";
        }
        if (StringUtils.isBlank(dto.getGender())) {
            return "性别不能为空";
        }
        if (!"男".equals(dto.getGender()) && !"女".equals(dto.getGender())) {
            return "性别格式错误，应填写：男 或 女";
        }
        if (StringUtils.isBlank(dto.getCollegeName())) {
            return "学院不能为空";
        }
        if (StringUtils.isBlank(dto.getTitle())) {
            return "职称不能为空";
        }
        if (!Arrays.asList("教授", "副教授", "讲师", "助教").contains(dto.getTitle())) {
            return "职称格式错误，应填写：教授、副教授、讲师、助教";
        }
        return null;
    }
}
```

## SQL查询示例

```sql
-- 1. 获取教师列表（带分页和筛选）
SELECT 
  t.teacher_id,
  t.user_id,
  t.school_id,
  t.name,
  t.employee_no,
  t.gender,
  t.college_name,
  t.title,
  t.expertise,
  t.intro,
  t.guidance_count,
  t.rating_score,
  t.phone,
  t.email,
  TO_CHAR(t.create_time, 'YYYY-MM-DD HH24:MI:SS') AS create_time
FROM biz_teacher t
WHERE t.school_id = #{schoolId}  -- 只查询本校教师
  AND (#{name} IS NULL OR t.name LIKE CONCAT('%', #{name}, '%'))
  AND (#{employeeNo} IS NULL OR t.employee_no LIKE CONCAT('%', #{employeeNo}, '%'))
  AND (#{collegeName} IS NULL OR t.college_name LIKE CONCAT('%', #{collegeName}, '%'))
  AND (#{title} IS NULL OR t.title LIKE CONCAT('%', #{title}, '%'))
ORDER BY t.create_time DESC
LIMIT #{pageSize} OFFSET #{offset};

-- 2. 统计总数
SELECT COUNT(*) 
FROM biz_teacher t
WHERE t.school_id = #{schoolId}
  AND (#{name} IS NULL OR t.name LIKE CONCAT('%', #{name}, '%'))
  AND (#{employeeNo} IS NULL OR t.employee_no LIKE CONCAT('%', #{employeeNo}, '%'))
  AND (#{collegeName} IS NULL OR t.college_name LIKE CONCAT('%', #{collegeName}, '%'))
  AND (#{title} IS NULL OR t.title LIKE CONCAT('%', #{title}, '%'));

-- 3. 获取教师详情
SELECT 
  t.teacher_id,
  t.user_id,
  t.school_id,
  t.name,
  t.employee_no,
  t.gender,
  t.college_name,
  t.title,

FROM biz_teacher t
```

### Java Controller示例

```java
@RestController
@RequestMapping("/api/school/teacher")
public class SchoolTeacherController {

    @Autowired
    private SchoolTeacherService teacherService;

    @GetMapping("/list")
    public Result<PageResult<TeacherVO>> getTeacherList(TeacherQuery query) {
        // 从token获取当前学校ID
        String schoolId = SecurityUtils.getSchoolId();
        query.setSchoolId(schoolId);
        
        PageResult<TeacherVO> result = teacherService.getTeacherList(query);
        return Result.success(result);
    }

    @GetMapping("/detail/{teacherId}")
    public Result<TeacherVO> getTeacherDetail(@PathVariable String teacherId) {
        String schoolId = SecurityUtils.getSchoolId();
        TeacherVO teacher = teacherService.getTeacherDetail(teacherId, schoolId);
        
        if (teacher == null) {
            return Result.error("教师不存在或无权访问");
        }
        
        return Result.success(teacher);
    }
}
```

## 注意事项

1. **权限控制**: 学校端只能查看本校教师数据，需要在SQL中加上 `school_id` 过滤条件
2. **数据隔离**: 从token中获取当前登录用户所属的学校ID，确保数据隔离
3. **性别字段**: 1=男, 2=女
4. **职称字段**: 教授、副教授、讲师、助教
5. **评分字段**: `rating_score` 是 NUMERIC(2,1) 类型，范围0.0-5.0
6. **指导数量**: `guidance_count` 初始值为0，后续可通过指导记录自动更新
7. **批量导入**:
   - 使用EasyExcel读取Excel文件
   - 需要同时创建 `sys_user` 和 `biz_teacher` 两张表的记录
   - 使用事务确保数据一致性
   - 初始密码为 `123456`，需要MD5加密
   - 工号作为登录账号（`login_identity`）
   - 性别转换：男=1，女=2
   - 导入失败的记录需要返回详细的失败原因
8. **模板下载**: 模板中应包含示例数据，方便用户理解填写格式
9. **文件大小限制**: 建议限制上传文件大小不超过5MB
10. **并发控制**: 如果导入数据量大，建议使用异步处理或分批导入
