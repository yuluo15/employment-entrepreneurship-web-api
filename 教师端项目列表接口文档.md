# 教师端项目列表接口文档

## 1. 获取教师项目列表

### 接口信息
- **接口路径**: `/api/teacher/projects`
- **请求方式**: `GET`
- **接口描述**: 获取教师可以查看和指导的项目列表，支持全网项目和本校项目切换、搜索、筛选、分页

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | Number | 是 | 页码，从1开始 |
| pageSize | Number | 是 | 每页数量，建议10 |
| keyword | String | 否 | 搜索关键词（项目名称、学生姓名） |
| scope | String | 否 | 范围：all=全网项目，school=本校项目，默认school |
| status | String | 否 | 项目状态：0=待审核，1=孵化中，2=已驳回，3=已落地 |
| domain | String | 否 | 项目领域：internet=互联网+，culture=文化创意，tech=科技创新，service=社会服务 |

### 请求头
```
Authorization: Bearer {token}
```

### 请求示例
```
GET /api/teacher/projects?pageNum=1&pageSize=10&scope=school&status=0
```

### 业务逻辑说明

1. **全网项目 vs 本校项目**
   - `scope=all`: 查询所有学校的项目
   - `scope=school`: 只查询当前教师所在学校的项目
   - 默认显示本校项目

2. **搜索逻辑**
   - 支持按项目名称模糊搜索
   - 支持按学生姓名模糊搜索

3. **筛选逻辑**
   - 可按项目状态筛选
   - 可按项目领域筛选
   - 多个筛选条件同时生效（AND关系）

4. **排序规则**
   - 待指导项目（guidanceCount=0）优先显示
   - 按创建时间倒序排列

### 数据库查询示例

```sql
-- 查询项目列表
SELECT 
  p.project_id,
  p.project_name,
  p.logo,
  p.student_id,
  u.nickname as student_name,
  p.school_id,
  s.school_name,
  p.college_name,
  p.domain,
  p.team_size,
  p.status,
  COALESCE(g.guidance_count, 0) as guidance_count,
  p.create_time
FROM project p
LEFT JOIN user u ON p.student_id = u.user_id
LEFT JOIN school s ON p.school_id = s.school_id
LEFT JOIN (
  SELECT project_id, COUNT(*) as guidance_count
  FROM guidance
  GROUP BY project_id
) g ON p.project_id = g.project_id
WHERE 1=1
  -- 如果 scope=school，只查询本校项目
  AND (#{scope} = 'all' OR p.school_id = #{teacherSchoolId})
  -- 关键词搜索
  AND (#{keyword} IS NULL OR p.project_name LIKE CONCAT('%', #{keyword}, '%') OR u.nickname LIKE CONCAT('%', #{keyword}, '%'))
  -- 状态筛选
  AND (#{status} IS NULL OR p.status = #{status})
  -- 领域筛选
  AND (#{domain} IS NULL OR p.domain = #{domain})
ORDER BY 
  CASE WHEN COALESCE(g.guidance_count, 0) = 0 THEN 0 ELSE 1 END,
  p.create_time DESC
LIMIT #{offset}, #{pageSize}

-- 查询总数
SELECT COUNT(*) as total
FROM project p
LEFT JOIN user u ON p.student_id = u.user_id
WHERE 1=1
  AND (#{scope} = 'all' OR p.school_id = #{teacherSchoolId})
  AND (#{keyword} IS NULL OR p.project_name LIKE CONCAT('%', #{keyword}, '%') OR u.nickname LIKE CONCAT('%', #{keyword}, '%'))
  AND (#{status} IS NULL OR p.status = #{status})
  AND (#{domain} IS NULL OR p.domain = #{domain})
```

### 错误码说明
| 错误码 | 说明 |
|--------|------|
| 401 | 未登录或token过期 |
| 403 | 无权限访问（非教师角色） |
| 500 | 服务器内部错误 |

### 注意事项
1. 该接口需要验证用户角色为教师（role = 'ROLE_TEACHER'）
2. 从token中解析出教师ID和学校ID
3. 时间字段需要转换为相对时间格式
4. 待指导项目（guidanceCount=0）需要优先显示
